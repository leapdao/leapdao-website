LeapDAO

Scaling for decentralized apps

The current capacity of Ethereum network is fairly limited in its transaction throughput. Leap Network is increasing this capacity and thus the scope of possible dApps in the ecosystem through the development of Plasma. This enables decentralized applications with thousands of users at the same time.

LeapDAO is open Decentralized Adaptive Organisation run by Holacracy principles with the goal to build a self-managing and purpose-driven community. Anyone can easily join us and start contribute in minutes. We use various reward options including bounties.
